# Mobile focus patch

Deze patch corrigeert de verhouding op smartphone.

## Probleem

De opgave/visualisatie was te klein, terwijl de numpad/keypad veel te groot was.

## Oplossing

- Vraagtekst groter.
- Visualisaties groter en met minder lege rand.
- Antwoordzone compacter.
- Keypad veel compacter.
- Bij lage telefoons worden extra hulpkoppen/onderknoppen ingeklapt.
- Geodriehoekmodus behoudt voldoende ruimte, maar zonder een gigantisch leeg vlak.
